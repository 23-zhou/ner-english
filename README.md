# ner-english
英文命名实体识别(NER)的研究

## 准备
数据集：Kaggle-https://www.kaggle.com/abhinavwalia95/entity-annotated-corpus/version/4

实体标签含义：
```
geo = Geographical Entity 地名
org = Organization 组织
per = Person 人物
gpe = Geopolitical Entity 地理政治
tim = Time indicator 时间
art = Artifact 艺术
eve = Event 时间
nat = Natural Phenomenon 自然现象
```
